Task 4 — Complex Form Validation & DOM Manipulation
------------------------------------------------------------

Features:
- Password strength indicator (weak → medium → strong)
- Confirm-password match check
- Real-time validation messages
- Interactive UI element that appears or hides on button click
- Dynamic DOM updates using JavaScript
- Responsive layout for both desktop and mobile

How to run:
1. Open index.html in your browser
(No backend or server required)

What this demonstrates:
- Practical client-side validation techniques
- DOM manipulation with JavaScript
- Interactive elements that respond to user actions
- Clean and responsive front-end behavior

Notes:
